# راه‌اندازی بات تلگرام — نسخهٔ فارسی

مراحل اولیه:
1. بات را در `@BotFather` بسازید و توکن را دریافت کنید.
2. مقدار `TELEGRAM_BOT_TOKEN` را در ` .env` قرار دهید.
3. `TELEGRAM_CHAT_ID` را مشخص کنید (چت شخصی یا کانال)

اجرای بات:

```bash
uv run main.py bot
```

دستورات متداول:
- `/apply <url> [--template altacv]` — تولید CV و نامه و دریافت خروجی در چت

نکته: برای کارکرد صحیح بات نیاز به اینترنت، API OpenAI (در صورت استفاده از LLM) و دسترسی به ابزارهای LaTeX روی ماشین وجود دارد.
