# معماری سیستم — نسخهٔ فارسی

این پروژه چند مولفهٔ اصلی دارد:

- `src/pipeline/` — لولهٔ پردازش که شامل استخراج آگهی (`job_scraper`)، دسته‌بندی نقش (`role_classifier`)، تولید محتوا‌ی متناسب (`content_tailor`) و تولید LaTeX/پی‌دی‌اف (`latex_builder`) است.
- `src/bot/` — رابط تلگرام برای ارسال دستور `/apply` و آرشیو کردن خروجی‌ها.
- `templates/` — قالب‌های LaTeX شامل یک پوشهٔ مشترک `templates/shared/` برای اطلاعات شخصی.
- `compile.sh` — اسکریپتی که قالب‌ها را پیدا و می‌سازد.

جریان کلی:
1. دریافت URL آگهی → استخراج متن آگهی
2. دسته‌بندی نقش (AI / IT / PhD)
3. تولید محتوا (نکات برجسته، بندهای قابل درج در CV و نامه)
4. پر کردن قالب‌های LaTeX و کامپایل
5. خروجی: دو PDF (CV و Cover Letter) و ذخیره در `Applied/` و `output/`.
