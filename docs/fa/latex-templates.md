# قالب‌های LaTeX — نسخهٔ فارسی

ساختار قالب‌ها:

- `templates/<template_name>/` — هر قالب LaTeX در پوشهٔ خودش قرار دارد.
- `templates/shared/personal_data.tex` و فایل JSON متناظر: اطلاعات تماس و پروفایل یک‌جا نگهداری می‌شوند تا در همه قالب‌ها قابل استفاده باشند.

ورود اطلاعات در قالب‌ها:
- فایل‌های CV با `\input{../shared/personal_data}` اطلاعات شخصی را بازخوانی می‌کنند.

کامپایل:
- از `compile.sh` برای پیدا کردن همه قالب‌ها و کامپایل آن‌ها استفاده کنید.
- برای نامه‌های پوششی از `xelatex` (برای پشتیبانی از فونت‌ها) و برای CVها معمولاً `pdflatex` کافی است.
