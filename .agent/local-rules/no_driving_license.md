# Règle absolue : Pas de permis de conduire

Le candidat **n'a pas de permis de conduire** et n'en aura pas dans un avenir prévisible.

## Règle de filtrage (NON-NÉGOCIABLE)

Avant de générer un CV ou une lettre de motivation, scanner le texte de l'offre pour :
- "Permis B obligatoire"
- "Permis de conduire obligatoire"
- "Permis B exigé"
- "Permis B requis"
- "Driving license required"
- "Valid driver's license required"
- toute formulation indiquant que le permis est **obligatoire** (pas seulement "souhaité")

Si une de ces phrases est détectée → **STOPPER immédiatement** et informer l'utilisateur :
> "⛔ Cette offre exige le Permis B. Le candidat n'ayant pas de permis, cette candidature est incompatible. Aucun CV généré."

## Distinction importante
- "Permis B souhaité" ou "Permis B apprécié" → acceptable, continuer
- "Permis B obligatoire" / "exigé" / "requis" → **BLOQUER**

## Pourquoi
Le candidat a perdu du temps à préparer un CV complet pour CC Vallée des Baux avant de réaliser que le Permis B était obligatoire (juin 2026). Cette vérification doit se faire en premier, avant tout traitement.
